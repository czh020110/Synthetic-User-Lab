# 常见命令

- `conda activate synthetic-user-lab` - 激活项目默认 Conda 环境。
- `python -m uvicorn backend.main:app --host 127.0.0.1 --port 8765` - Windows 下验证 FastAPI / Playwright demo run 的推荐启动命令。
- `python -m uvicorn backend.main:app --host 127.0.0.1 --port 8765 --reload` - 普通 FastAPI 调试可用；Windows 下验证 Playwright demo run 不要使用该命令。
- `D:/Env/Anaconda/Anaconda3-2024.06-1/envs/synthetic-user-lab/python.exe <script>` - Windows 下脚本需要稳定输出中文 JSON 时优先直接调用环境 Python，避免 `conda run` 的 GBK 转码问题。
- `python -m playwright install` - 在 `synthetic-user-lab` 环境中安装 Playwright 浏览器驱动。
- 复杂异步或 HTTP 轮询验证 - 不要塞进 `python -c` 单行命令，优先 heredoc、临时脚本或直接环境 Python 执行。
