# 关键环境说明文档

## 1. 推荐开发环境

项目必须使用 `synthetic-user-lab` Conda 环境运行。

环境位置：

```text
D:\Env\Anaconda\Anaconda3-2024.06-1\envs\synthetic-user-lab
```

当前 Python 版本：

```text
Python 3.10.19
```

代码开发IDE:vscode

VSCode 调试启动 FastAPI 推荐端口：

```text
8765
```

说明：优先使用 `8765` 端口进行启动与验证，避免 8001 在部分 Windows 环境下触发套接字权限错误。

## 2. FastAPI 关键包

关于FastAPI核心包：

```text
fastapi==0.135.3
aiomysql==0.3.2
aiosqlite==0.22.1
pymysql==1.1.2
```

已保留 `synthetic-user-lab` 中原有版本的相关依赖：

```text
uvicorn==0.41.0
sqlalchemy==2.0.48
starlette==1.0.0
pydantic==2.12.5
```

说明：`PyMySQL` 安装元数据版本为 `1.1.2`，但模块内部 `__version__` 可能显示旧兼容字符串，不以该字符串作为实际安装版本判断依据。

## 3. 项目关键技术方向

当前环境主要服务于 Synthetic User Lab 的第一阶段开发：

- FastAPI：后端 API 服务
- Pydantic：接口与任务数据结构
- LangGraph：多节点任务编排
- LangChain：工具封装、Retriever 与上下文组织
- Playwright：Web 页面执行器，后续需要单独安装浏览器驱动
- Chroma / Redis：检索与运行状态相关能力
- OpenAI / Anthropic / DashScope / Ollama：多模型接入实验

## 4. 注意事项

- 本项目后续开发默认使用 `synthetic-user-lab` 环境

- 如果后续引入 Playwright，需要在该环境中安装浏览器。
