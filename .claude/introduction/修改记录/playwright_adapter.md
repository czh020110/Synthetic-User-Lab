# playwright_adapter

## 修复浏览器会话创建失败时缺少关键日志的问题(最新修改)

- 修改的文件名和路径(单个文档只写一次):
  - `backend/execution/playwright_adapter.py`

- 修改前存在的问题:
  - 浏览器会话创建失败时缺少明确日志，手动测试只能看到 run failed，无法快速定位初始化阶段问题。

- 添加前未完成的功能:
  - 浏览器会话创建失败的关键日志输出未实现。

- 如何修复的(关键修改点说明):
  - 增加 `logger` 并在会话创建开始时记录 `headless` 信息。
  - 在会话创建异常分支使用 `logger.exception(...)` 输出完整失败栈。

- 修改后的预期功能或修复后的预测结果:
  - 当会话初始化失败时可在服务日志直接看到失败栈，便于快速定位环境或浏览器问题。
