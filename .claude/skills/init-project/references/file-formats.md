# 目标文件格式模板

以下格式由 `update-docs` agent 在写入时严格遵守，本文件供 `init-project` skill 按需引用。

## 目标边界.md 格式

```md
# 目标边界

## 总体目标

[总体目标]

## 当前阶段目标

- [ ] [目标]：验收方式：[如何证明完成]

## 成功标准

- [ ] [可验证标准]

## 非目标

- [不属于当前阶段的内容]

## 质量底线

- 测试底线：[要求]
- 安全底线：[要求]
- 性能底线：[要求/当前暂无]
- 文档底线：[要求]

## 推进节奏

- [阶段推进规则]
```

## 项目目标.md 格式

```md
# 项目目标

## 总体流程目标

[当前项目最终需要实现的完整流程大方向]

## 核心功能目标

- [ ] [需要实现的功能或能力]：验收方式：[如何证明完成]

## 阶段功能范围

- [阶段/模块]：目标：[本阶段要完成什么]；状态：[未开始/进行中/完成]；依赖：[依赖/无]

## 实现发现

- 暂无。

## 后续目标调整

- 暂无。
```

## STEP.md 格式

```md
# STEP

## 长期方向

- [ ] S-001：[项目大方向步骤]
  - 目标：[长期目标]
  - 完成标准：[如何判断这个大方向完成]
  - 影响范围：[影响哪些模块/阶段]
```
