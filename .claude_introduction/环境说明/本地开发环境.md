# 本地开发环境

## 必需工具

- Conda：已安装；检查命令：`conda --version`
- Python：3.10.19（位于 `synthetic-user-lab` 环境中）；检查命令：`python --version`
- Playwright：需在 `synthetic-user-lab` 环境中安装浏览器驱动；检查命令：`python -m playwright --version`
- VSCode：当前默认开发 IDE；检查命令：`code --version`

## 初始化步骤

```bash
conda activate synthetic-user-lab
python -m playwright install
```

## 环境变量

- `MODEL_PROVIDER`：必需：是；说明：选择当前使用的模型供应商；示例：`openai`
- `OPENAI_MODEL_NAME`：必需：否；说明：决策 agent 与验证 agent 使用的主模型；示例：`gpt-4o`
- `OPENAI_FAST_MODEL_NAME`：必需：否；说明：等待观察 / 普通 wait agent 使用的快速模型，未设置时默认回退 `gpt-5.4-mini`；示例：`gpt-5.4-mini`
- `OPENAI_BASE_URL`：必需：否；说明：OpenAI 兼容接口地址；示例：`https://api.openai.com/v1`
- `OPENAI_API_KEY`：必需：否；说明：OpenAI 模型调用密钥；示例：`sk-***`
- `DASHSCOPE_MODEL_NAME`：必需：否；说明：DashScope 主模型名称；示例：`qwen3.5-flash`
- `DASHSCOPE_FAST_MODEL_NAME`：必需：否；说明：DashScope 等待观察 / 普通 wait 快速模型，未设置时默认回退 `qwen3.5-flash`；示例：`qwen3.5-flash`
- `DASHSCOPE_BASE_URL`：必需：否；说明：DashScope 自定义接口地址；示例：`https://dashscope.aliyuncs.com/compatible-mode/v1`
- `DASHSCOPE_API_KEY`：必需：否；说明：DashScope 模型调用密钥；示例：`sk-***`
- `SYNTHETIC_USER_LAB_HEADLESS`：必需：否；说明：浏览器默认是否无头运行；示例：`true`
- `SYNTHETIC_USER_LAB_RUN_STEP_LIMIT`：必需：否；说明：demo run 默认最大步数；示例：`8`
- `SYNTHETIC_USER_LAB_API_PREFIX`：必需：否；说明：FastAPI 路由前缀；示例：`/api/v1`
- `RAG_EMBEDDING_BASE_URL`：必需：否；说明：query-project 向量检索使用的 embedding 服务地址；示例：`https://dashscope.aliyuncs.com/compatible-mode/v1`
- `RAG_EMBEDDING_API_KEY`：必需：否；说明：query-project embedding 服务密钥；示例：`sk-***`
- `RAG_EMBEDDING_MODEL`：必需：否；说明：query-project embedding 模型 ID；示例：`text-embedding-v4`
- `RAG_RERANK_BASE_URL`：必需：否；说明：query-project 重排服务地址；示例：`https://dashscope.aliyuncs.com/compatible-api/v1`
- `RAG_RERANK_API_KEY`：必需：否；说明：query-project rerank 服务密钥；示例：`sk-***`
- `RAG_RERANK_MODEL`：必需：否；说明：query-project rerank 模型 ID；示例：`qwen3-rerank`

## 外部服务

- OpenAI 兼容模型服务：用途：为 decide / validate / wait observe agent 提供模型调用；启动/连接方式：通过 `.env` 中的 `OPENAI_*` 或兼容供应商地址配置。
- DashScope：用途：为 decide / validate / wait observe agent 提供模型调用；启动/连接方式：通过 `.env` 中的 `DASHSCOPE_*` 变量配置。
- query-project RAG embedding / rerank 服务：用途：为 `query-project` 文档检索提供向量化与重排；启动/连接方式：通过 `.claude/settings.local.json` 或当前 shell 环境注入 `RAG_EMBEDDING_*` 与 `RAG_RERANK_*` 变量。
- Playwright 浏览器驱动：用途：执行 demo 页面真实交互与截图；启动/连接方式：在 `synthetic-user-lab` 环境中运行 `python -m playwright install`。

## 已知环境坑

- Windows 默认端口冲突：表现：部分环境使用 `8001` 启动 FastAPI 时会触发套接字权限错误；处理：优先使用 `8765` 端口启动与验证。
- `conda run` 中文 JSON 输出乱码：表现：脚本输出中文 JSON 时可能出现 GBK 转码问题；处理：优先直接调用 `D:/Env/Anaconda/Anaconda3-2024.06-1/envs/synthetic-user-lab/python.exe`。
- Playwright 浏览器未安装：表现：初始化浏览器会话时报缺少浏览器驱动；处理：在 `synthetic-user-lab` 环境中执行 `python -m playwright install`。
- 未配置快速模型变量：表现：普通 `wait` 与等待观察节点会回退到供应商默认快速模型；处理：如需固定等待链路模型，显式设置 `OPENAI_FAST_MODEL_NAME` 或 `DASHSCOPE_FAST_MODEL_NAME`。
- 项目事实目录重命名后未同步 query-project 状态：表现：`query-project` 检索结果为空、返回旧目录结果，或 SKILL 描述未显示 `(rag已配置)`；处理：确认项目事实文档位于仓库根目录 `.claude_introduction/`，先执行 `python .claude/skills/query-project/scripts/query_introduction_rag.py --sync-config-status`，状态正常后再执行 `python .claude/skills/query-project/scripts/query_introduction_rag.py --refresh-index` 刷新索引。
- RAG embedding / rerank 服务不可达：表现：执行 `query_introduction_rag.py --refresh-index` 时返回 `CONFIG_ERROR[embedding]` 或 `CONFIG_ERROR[rerank]`；处理：先确认 `.claude/settings.local.json` 或当前 shell 中的 `RAG_EMBEDDING_*`、`RAG_RERANK_*` 配置正确，并确保对应服务网络可达后再重试刷新。
